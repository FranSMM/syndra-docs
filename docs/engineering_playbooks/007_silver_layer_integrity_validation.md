## Playbook 7: Silver Layer Integrity Validation (Data Profiling)

To confirm that the data generated by the MLOps pipeline meets institutional B2B standards, the Data Engineer must execute this profiling query weekly:

1. Execute Quant Profiling:

```bash
docker compose exec postgres psql -U syndra_user -d syndra_db -c "
  SELECT 
    payload->'financial_metadata'->>'sentiment_label' as label,
    payload->'financial_metadata'->>'sentiment_score' as score,
    COUNT(*)
  FROM raw_articles 
  WHERE ingestion_state = 'processed'
  GROUP BY label, score
  ORDER BY COUNT(*) DESC
  LIMIT 10;
"
```

2. Acceptance Criteria (SLA):

* There must be no null tags (label is NULL).
* The score variance in the neutral category must present values other than an absolute 0.0, demonstrating the mathematical sensitivity of the FinBERT model.
